Getting Started

This script works on Windows operating systems. You need Administrator privileges to access all system details. The script utilizes WMIC (Windows Management Instrumentation Command-line).

How to Use

  1. Press Ctrl + R on your keyboard to open the Run dialog box.
  2. Type cmd and press Enter to open the Command Prompt.
  3. Right-click on the Command Prompt icon and select "Run as Administrator" for full access.
  4. Click the button below to copy the command.

System Information Extracted

  • CPU Information: CPU Name, Maximum Clock Speed, Architecture, Number of Physical Cores, Logical Processors, Voltage, L2 and L3 Cache Sizes.
  • Memory Information: RAM Capacity, Speed, Manufacturer, Part Number.
  • Disk Information: Disk Model, Size, Status, Partitions, Plug and Play Device ID, Volume Name, Serial Number.
  • BIOS Information: BIOS Release Date, Serial Number, Version, Current BIOS Voltage.
  • Graphics Information: GPU Name, Adapter RAM, Driver Version, Video Processor.
  • Battery Information: Battery Name, Charge Percentage, Design Capacity, Full Charge Capacity, Expected Battery Life, Time to Full Charge.
  • Network Information: Adapter Name, Speed, MAC Address, Packets Sent/Received.
  • USB Information: USB Device ID, Manufacturer, Status.
  • Hotfix and Update Information: Hotfix ID, Description, Installed On (Date).
  • Process Information: Process Name, ID, Thread Count, Memory Usage.
  • Miscellaneous: System Manufacturer, Model, Type, Last Boot Time, Boot Device, Thermal State.

Redirecting Output

To save the output to a text file, run the following command:

scriptname.bat > systeminfo.txt
